== Description ==

Built by designers, for designers. Add SVG images with ease, via a single function or shortcode. Comes with icons from icomoon!  The best part of this plugin is that it doesn't load any icons unless you use them on the page, keeping page size down. Compliments of Big Sea Inc.

== Installation ==

1. Upload the files to the /wp-content/plugins/bsd-svg-icons/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Start using the icons!
